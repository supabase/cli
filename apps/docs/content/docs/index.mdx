The Supabase CLI lets you develop locally, manage your Supabase projects, and
deploy from the terminal or CI/CD.

## Quick start

```sh
# Install
brew install supabase/tap/supabase   # macOS
npx supabase                          # or via npx

# Authenticate
supabase login

# Start a project
supabase init
supabase start
```
